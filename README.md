# Optimal-Stopping

A starter repository for Optimal Stopping experiments and implementations.

## Contents

- src/optimal_stopping: Python package skeleton
- tests: basic test suite
- .github/workflows/ci.yml: GitHub Actions CI for Python

## Quickstart

1. Create a Python virtual environment: python -m venv .venv
2. Activate it and install dependencies: pip install -e .[test]
3. Run tests: pytest
